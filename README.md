# Freecad_github_mcp

This repository provides tools, scripts, and guides for integrating FreeCAD with GitHub and VS Code using the Model Context Protocol (MCP).

## Features
- Example scripts for FreeCAD automation
- Installation guides in English, Japanese, and Chinese
- Resources for setting up FreeCAD-MCP in VS Code

## Getting Started
1. Clone this repository.
2. Follow the installation guide (`FreeCAD-MCP-VSCode-Installation-Guide.md`) for setup instructions.
3. Explore the `examples/` directory for sample scripts.
